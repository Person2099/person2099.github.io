# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability in this website, please report it responsibly by creating an issue on this repo using the security label or using the contact form.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

I will acknowledge your report within 48 hours and work to address the issue promptly.

## Third-Party Dependencies

- **Google Fonts**: Subresource Integrity enabled
- **Icons**: SVGs embedded or from CDN with SRI

## Security Updates

Dependencies are kept up-to-date through:

- GitHub Dependabot alerts
- Manual security reviews
- Regular dependency audits
